# jordyreins.nl

This is the source code for my personal (portfolio) website, [jordyreins.nl](https://jordyreins.nl).

## Developing

If you want to check out the website locally, you can run the following command:

```sh
bun install

bun run dev

# or start the server and open the app in a new browser tab
bun run dev -- --open
```

## Building

To build the website to a Docker container, run the following command:

```sh
docker compose up -d
```
